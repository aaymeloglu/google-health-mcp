# Discovery Kit

Use this copy for MCP directories, GitHub lists, launch posts and community replies.

## One-line Description

Unofficial local-first Google Health API v4 MCP server for AI agents, with OAuth tokens stored on the user's machine.

## Longer Description

Google Health MCP lets Claude, Cursor, Codex, Hermes, OpenClaw and other MCP clients query user-authorized Google Health API v4 context through scoped tools instead of raw credentials. It is designed for Fitbit, Pixel Watch and supported Google Health data sources, with read-only defaults, local token storage, privacy audit tools and explicit beta boundaries while the API stabilizes.
